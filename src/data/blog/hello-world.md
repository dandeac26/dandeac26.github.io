# Hello World!

Welcome to my new portfolio website, freshly rebuilt with **React**, **TypeScript**, and **Tailwind CSS**.

## Why the rewrite?

The original site was built using the [Prologue](https://html5up.net/prologue) template from HTML5 UP — a great starting point, but as my projects grew, I wanted:

- A **modern component-based architecture** for easier maintenance
- **Dark mode** support (because who doesn't love dark mode?)
- A **blog section** to share my thoughts and learnings
- **Better animations** with Framer Motion
- A proper **build pipeline** with TypeScript type safety

## Tech Stack

- **Vite** — blazing fast HMR and build
- **React 18** — UI components
- **TypeScript** — type safety everywhere
- **Tailwind CSS v4** — utility-first styling
- **Framer Motion** — smooth animations
- **Lucide React** — clean, consistent icons
